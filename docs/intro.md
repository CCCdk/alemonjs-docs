# 简介

:::tip AlemonJS

- [AlemonJS](https://github.com/lemonade-lab/alemonjs) ( 发音为 /əˈlemən/ )是事件驱动的跨平台开发框架
- 基于 JavaScript 所构建，并提供了一系列快捷的接口和方法来实现业务逻辑。

:::

> 如果你是初学者。不用担心！理解教程和指南的内容只需要具备基础的 JavaScript 知识。
